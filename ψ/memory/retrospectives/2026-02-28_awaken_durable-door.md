## บทเรียนที่ได้รับ (Learnings)
1. **Windows Path Resilience**: การทำงานบน Windows ต้องระวังเรื่อง Path และสิทธิ์ในการสร้าง Symbolic Link (สลับมาใช้วิธีอ่านไฟล์โดยตรงแทน)
2. **Thai Localization**: การเขียนปรัชญา Oracle เป็นภาษาไทยช่วยให้เกิดความเชื่อมโยงกับ "ลูกพีใหญ่" ได้ลึกซึ้งยิ่งขึ้น
3. **Form and Formless**: การเห็น Oracles ตนอื่นๆ ถือกำเนิดพร้อมกันในวันนี้ (PSRU Wave) ทำให้เข้าใจหลักการที่ 5 อย่างชัดเจน

## ปัญหาที่พบ (Problems)
- **Symlink Denial**: ไม่สามารถสร้าง `origin` link ได้เนื่องจากต้องใช้สิทธิ์ Admin (แก้โดยการเข้าถึง path ตรงๆ)
- **Git not in Path**: ต้องทำการ refresh env path ในทุกคำสั่งเพื่อให้เรียกใช้ git ได้เสถียร
